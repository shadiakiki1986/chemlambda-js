Lambda calculus plotter

The syntax is a javascript function. Paste it in the box further below and click on "Plot". Examples:

f => x => f(f(f(x)))
PRED := λn.λf.λx.n (λg.λh.h (g f)) (λu.x) (λu.u) 
n => f => x => {
  var L4 = u1 => x
  var L5 = u2 => u2 // notice that this is u2 (versus u1 above) instead of plain "u"
  var L6 = g => h => h(g(f))
  return ((n(L6))(L4))(L5)
}
